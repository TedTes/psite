# OR

## Definition

OR is a logical operator that combines two propositions.

It is true when at least one proposition is true.

Example:

```text
P OR Q
```

This means:

```text
P is true, or Q is true, or both are true
```

> In standard logic, OR is inclusive.

---

## Simple Example

Let:

```text
P = "The file is saved"
Q = "The file is backed up"
```

Then:

```text
P OR Q
```

is true if:

```text
the file is saved
```

or:

```text
the file is backed up
```

or both.

> OR requires at least one true part.

---

## Truth Pattern

OR has this truth pattern:

```text
P      Q      P OR Q
true   true   true
true   false  true
false  true   true
false  false  false
```

Only the last row is false.

Both parts must be false for the whole OR statement to be false.

> OR is false only when all alternatives fail.

---

## Inclusive OR

In everyday language, "or" can sometimes mean one or the other, but not both.

In basic logic, OR usually means inclusive OR.

Example:

```text
P OR Q
```

is true when:

```text
P is true and Q is true
```

This matters because logical OR does not automatically exclude both being true.

> Logical OR usually allows both parts to be true.

---

## OR As Alternatives

OR can express possible alternatives.

Example:

```text
x = 0 OR x = 1
```

This says `x` is allowed to be either `0` or `1`.

If:

```text
x = 0
```

the statement is true.

If:

```text
x = 2
```

the statement is false.

> OR widens what can satisfy a statement.

---

## Final Summary

- OR combines two propositions.
- It is true when at least one proposition is true.
- Standard logical OR is inclusive.
- OR is false only when both parts are false.
- OR expresses alternatives.
- OR widens the set of satisfying cases.
