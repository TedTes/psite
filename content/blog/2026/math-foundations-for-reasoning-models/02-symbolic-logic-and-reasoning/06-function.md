# Function

## Definition

A function maps input values to output values.

Example:

```text
f(x) = x + 1
```

If:

```text
x = 5
```

then:

```text
f(5) = 6
```

> Function = rule-governed mapping from input to output.

---

## Function Name And Input

In:

```text
f(x)
```

`f` is the function name.

`x` is the input placeholder.

The expression says:

```text
apply function f to input x
```

If:

```text
f(x) = x + 1
```

then `f` names the rule:

```text
add 1
```

> A function symbol names a rule for producing output.

---

## Function vs Value

A function is not one value.

It is a mapping that can produce values.

Example:

```text
f(x) = x + 1
```

Outputs:

```text
f(1) = 2
f(2) = 3
f(3) = 4
```

The function stays the same.

The output changes with the input.

> A function is a rule; its outputs depend on inputs.

---

## Function In Logic

In logic, a function can build a new term from an existing term.

Example:

```text
father(x)
```

This can mean:

```text
the father of x
```

If:

```text
x = Ada
```

then:

```text
father(Ada)
```

refers to a specific object under the interpretation.

> Logical functions create terms from other terms.

---

## Function vs Predicate

A function returns a value or object.

Example:

```text
double(3) = 6
```

A predicate returns a truth value.

Example:

```text
Even(3) = false
```

This distinction matters because functions produce terms, while predicates produce statements that can be true or false.

> Function returns an object or value; predicate returns true or false.

---

## Final Summary

- A function maps inputs to outputs.
- A function name represents a rule.
- The same function can produce different outputs for different inputs.
- In logic, functions can build terms.
- A function is different from a predicate.
- Functions help connect symbolic structure to rule-governed transformation.
