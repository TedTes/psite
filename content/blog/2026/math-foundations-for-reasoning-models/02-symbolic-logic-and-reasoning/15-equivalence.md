# Equivalence

## Definition

Equivalence means two propositions have the same truth value.

Basic form:

```text
P if and only if Q
```

This means:

```text
P is true exactly when Q is true
P is false exactly when Q is false
```

> Equivalence says two propositions match in truth value.

---

## Simple Example

Example:

```text
A number is even if and only if it is divisible by 2.
```

Let:

```text
P = number is even
Q = number is divisible by 2
```

Then:

```text
P if and only if Q
```

The two statements are true together or false together.

> Equivalent statements stand or fall together.

---

## Truth Pattern

Equivalence has this truth pattern:

```text
P      Q      P iff Q
true   true   true
true   false  false
false  true   false
false  false  true
```

It is true when both sides match.

It is false when they differ.

> Equivalence checks whether truth values are the same.

---

## Equivalence As Two Implications

Equivalence can be understood as two implications.

```text
P iff Q
```

means:

```text
If P, then Q
If Q, then P
```

Both directions must hold.

If only one direction holds, the statements are not equivalent.

> Equivalence requires implication in both directions.

---

## Equivalence vs Equality

Equivalence is not always the same as equality.

Equality usually says two values or objects are the same.

Example:

```text
x = 5
```

Equivalence says two propositions have the same truth behavior.

Example:

```text
P iff Q
```

> Equality compares values; equivalence compares truth behavior.

---

## Final Summary

- Equivalence means two propositions have the same truth value.
- It is often read as "if and only if."
- It is true when both propositions match.
- It is false when their truth values differ.
- Equivalence can be understood as implication in both directions.
- Equivalence is different from equality.
