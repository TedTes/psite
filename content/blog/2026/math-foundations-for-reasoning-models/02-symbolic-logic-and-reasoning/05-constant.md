# Constant

## Definition

A constant is a symbol with a fixed reference in a given system.

Example:

```text
5
```

The symbol `5` refers to a specific number.

Another example:

```text
Socrates
```

In a logical system, this may refer to one particular person.

> A constant is a symbol whose reference is fixed in the current interpretation.

---

## Constant vs Variable

A variable can vary.

Example:

```text
x
```

It may refer to different values depending on assignment.

A constant is fixed.

Example:

```text
5
```

It does not become `7` in the same interpretation.

> Variable = flexible reference. Constant = fixed reference.

---

## Constants In Expressions

Constants can appear inside expressions.

Example:

```text
x + 5
```

Here:

```text
x = variable
5 = constant
```

If `x = 2`, then:

```text
x + 5 = 7
```

The variable changes by assignment.

The constant remains fixed.

> Constants provide stable pieces inside expressions.

---

## Constants In Logic

In logic, a constant can name a specific object.

Example:

```text
Human(Socrates)
```

Here `Socrates` is a constant symbol.

It refers to a specific object in the interpretation.

Compare:

```text
Human(x)
```

Here `x` is a variable.

It can stand for different objects.

> Logical constants name specific objects.

---

## Fixed Within A Context

A constant is fixed within a system or interpretation.

That does not mean the same mark must have the same meaning in every possible system.

Example:

```text
pi
```

In mathematics, this usually refers to a specific constant.

But a badly defined system could use the same letters differently.

What matters is consistency inside the current interpretation.

> Constant reference is fixed relative to a context.

---

## Final Summary

- A constant is a symbol with fixed reference in a system.
- A variable can vary; a constant stays fixed.
- Constants can appear inside expressions.
- In logic, constants can name particular objects.
- Constant meaning depends on the interpretation but must stay consistent within it.
