# Logical Operators

## Definition

Logical operators combine or modify propositions.

They create new propositions from existing ones.

Examples:

```text
P AND Q
P OR Q
NOT P
If P, then Q
```

> Logical operators build compound propositions.

---

## Why Operators Matter

A single proposition can say one thing.

Example:

```text
P = "It is raining"
```

Logical operators let us build more structured statements.

Example:

```text
P AND Q
```

This says both `P` and `Q` hold.

Another example:

```text
NOT P
```

This says `P` does not hold.

> Operators let reasoning express relationships between statements.

---

## Operators Use Truth Values

Logical operators work on truth values.

Example:

```text
P = true
Q = false
```

Then:

```text
P AND Q = false
P OR Q = true
NOT P = false
```

The operator defines how the output truth value is produced.

> A logical operator is a rule for truth values.

---

## Common Operators

The basic operators are:

```text
AND
OR
NOT
IMPLIES
EQUIVALENT
```

Each operator has a specific rule.

Example:

```text
AND is true only when both parts are true.
```

Example:

```text
OR is true when at least one part is true.
```

> Each logical operator defines a different truth pattern.

---

## Operators Create Structure

Logical operators can be nested.

Example:

```text
(P AND Q) OR R
```

This expression has structure:

```text
first combine P and Q
then combine the result with R
```

Parentheses help make the structure clear.

> Logical operators build larger statements from smaller statements.

---

## Final Summary

- Logical operators combine or modify propositions.
- They produce new propositions.
- They operate on truth values.
- Common operators include AND, OR, NOT, implication, and equivalence.
- Operators define truth patterns.
- Operators allow complex reasoning structures to be built from simple statements.
