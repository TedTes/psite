# Truth Value

## Definition

A truth value is the logical status of a proposition.

In basic logic, the two truth values are:

```text
true
false
```

Example:

```text
2 + 3 = 5
```

Truth value:

```text
true
```

Example:

```text
2 + 3 = 9
```

Truth value:

```text
false
```

> Truth value tells whether a proposition is true or false.

---

## Truth Value Requires A Proposition

Only propositions have truth values.

Proposition:

```text
It is raining.
```

This can be true or false.

Not a proposition:

```text
Close the door.
```

This is a command.

It does not have a truth value.

> Truth values apply to statements, not every expression.

---

## Truth Value Depends On Interpretation

Some propositions depend on context.

Example:

```text
x > 5
```

If:

```text
x = 10
```

then:

```text
true
```

If:

```text
x = 2
```

then:

```text
false
```

The symbolic form is the same.

The truth value changes with interpretation.

> Truth value depends on what the symbols refer to.

---

## Truth Values In Compound Statements

Logical operators combine truth values.

Example:

```text
P AND Q
```

If:

```text
P = true
Q = false
```

then:

```text
P AND Q = false
```

The compound truth value is determined by the operator and the truth values of the parts.

> Compound statements compute truth values from smaller statements.

---

## Truth vs Validity

Truth applies to propositions.

Validity applies to arguments.

Example proposition:

```text
2 + 3 = 5
```

This is true.

Example argument form:

```text
If P, then Q
P
Therefore Q
```

This is valid.

> Truth belongs to statements; validity belongs to reasoning structure.

---

## Final Summary

- A truth value is true or false.
- Propositions have truth values.
- Commands and questions do not have truth values.
- Truth value can depend on interpretation.
- Logical operators build compound truth values.
- Truth is different from validity.
