# Premise

## Definition

A premise is a statement used as a starting point in an argument.

Example:

```text
Premise 1: If it rains, the ground gets wet.
Premise 2: It rains.
```

These statements are given.

Reasoning uses them to reach a conclusion.

> Premise = starting statement for reasoning.

---

## Premise vs Conclusion

A premise is what reasoning starts from.

A conclusion is what reasoning arrives at.

Example:

```text
Premise 1: If P, then Q
Premise 2: P
Conclusion: Q
```

The premises support the conclusion.

The conclusion is not just listed randomly.

It is supposed to follow from the premises.

> Premises provide support; conclusion is the result.

---

## Premises Can Be True Or False

A premise is not automatically true just because it is used in an argument.

Example:

```text
Premise: All birds can fly.
```

This premise is false because some birds cannot fly.

But we can still analyze the argument's structure.

> A premise is assumed for the argument, but it may still be false.

---

## Premises In Symbolic Form

Premises can be represented symbolically.

Example:

```text
If P, then Q
P
```

These are premises.

From them, we may infer:

```text
Q
```

Symbolic form helps us focus on the reasoning structure.

> Premises can be written as logical expressions.

---

## Why Premises Matter

Reasoning is only as strong as its starting points and structure.

If the premises are true and the reasoning is valid, the conclusion is secure.

If a premise is false, the conclusion may not be reliable.

> Premises are the input side of an argument.

---

## Final Summary

- A premise is a starting statement in an argument.
- Premises support a conclusion.
- A premise can be true or false.
- Symbolic logic can represent premises formally.
- Premises are different from conclusions.
- Reasoning depends on both premises and inference rules.
