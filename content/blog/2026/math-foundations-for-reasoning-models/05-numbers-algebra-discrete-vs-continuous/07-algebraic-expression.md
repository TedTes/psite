# Algebraic Expression

## Definition

An algebraic expression combines numbers, variables, and operations.

Examples:

```text
x + 1
2 * y
3x + 5
(a + b) / 2
```

> Algebraic expression = structured combination of quantities.

---

## Parts Of An Expression

An expression can contain:

- constants
- variables
- operators
- grouping

Example:

```text
3x + 5
```

Parts:

```text
3 is a coefficient
x is a variable
+ is an operator
5 is a constant
```

> Expressions are built from smaller symbolic parts.

---

## Expression vs Value

An expression is not always a single known value yet.

Example:

```text
x + 1
```

If:

```text
x = 3
```

then:

```text
x + 1 = 4
```

Before `x` is assigned, the expression describes a relationship.

> An expression can represent a value before the value is known.

---

## Expression vs Equation

An expression does not claim equality by itself.

Expression:

```text
x + 1
```

Equation:

```text
x + 1 = 4
```

The equation states that two expressions are equal.

> Expressions can be parts of equations.

---

## Why Expressions Matter

Expressions let us describe rules compactly.

Example:

```text
new value = old value + 1
```

The expression:

```text
old value + 1
```

describes how the next value is produced.

> Algebraic expressions give symbolic form to numerical rules.

---

## Final Summary

- An algebraic expression combines numbers, variables, and operations.
- Expressions can contain constants, variables, operators, and grouping.
- An expression may represent an unknown value.
- An expression is not the same as an equation.
- Expressions describe numerical structure and rules.
