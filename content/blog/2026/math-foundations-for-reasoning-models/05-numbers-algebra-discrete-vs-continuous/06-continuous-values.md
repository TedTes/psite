# Continuous Values

## Definition

Continuous values can vary through intermediate values.

Example:

```text
2.0
2.1
2.01
2.001
```

Between two continuous values, more values can exist.

> Continuous value = value that can vary smoothly within a range.

---

## Simple Examples

Continuous quantities often come from measurement.

Examples:

```text
height
weight
distance
temperature
time
position
```

A distance might be:

```text
2 meters
2.5 meters
2.57 meters
```

> Continuous values support fine-grained measurement.

---

## Continuous vs Discrete

Discrete:

```text
number of nodes = 4
```

Continuous:

```text
distance = 4.37 meters
```

You usually cannot have `4.37` nodes.

But you can measure `4.37` meters.

> Discrete counts separate units; continuous measures variable quantities.

---

## Real Numbers And Continuity

Real numbers are used to model continuous values.

Example:

```text
position on a line
```

The position can be:

```text
0
0.5
0.51
0.511
```

The model allows increasingly precise positions.

> Real numbers give a mathematical language for continuity.

---

## Computation And Approximation

Computers usually store continuous values approximately.

Example:

```text
temperature = 21.438
```

The stored value has limited precision.

The real-world quantity may vary more finely than the stored representation.

> Continuous quantities often become finite approximations inside computation.

---

## Final Summary

- Continuous values can vary through intermediate values.
- They are common in measurement.
- Real numbers model continuous values.
- Discrete values count separate options.
- Computers usually approximate continuous values with finite representations.
