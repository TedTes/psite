# Ordered Pair

## Definition

An ordered pair is a two-part structure where order matters.

Example:

```text
(a, b)
```

The first position is `a`.

The second position is `b`.

> Ordered pair = two objects with position.

---

## Order Matters

These are not generally the same:

```text
(a, b)
(b, a)
```

Example:

```text
(person, city)
```

means something different from:

```text
(city, person)
```

The same two objects appear, but their roles changed.

> Position gives meaning in an ordered pair.

---

## Ordered Pair vs Set

A set does not care about order.

Example:

```text
{a, b} = {b, a}
```

An ordered pair does care about order.

Example:

```text
(a, b) is not the same as (b, a)
```

> Sets ignore order; ordered pairs preserve order.

---

## Ordered Pairs In Relations

Relations are built from ordered pairs.

Example:

```text
R = {(1, 2), (2, 3), (3, 4)}
```

Each pair records a connection.

For example:

```text
(1, 2)
```

may mean:

```text
1 points to 2
```

> Ordered pairs are the basic units of relations.

---

## Final Summary

- An ordered pair has two positions.
- Order matters in an ordered pair.
- `(a, b)` is generally different from `(b, a)`.
- Ordered pairs are different from sets.
- Relations are sets of ordered pairs.
